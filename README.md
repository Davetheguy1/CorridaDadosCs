# CorridaDadosCs
## Descrição
Um jogo no qual o usuário compete contra a maquina em uma corrida, feito para a Academia do Programador.
## Funcionalidade
Após escolher entre 3 distâncias especificas, o usuário deve competir contra o computador em uma corrida na qual o deslocamento é ditado por um dado.
